模型压缩笔记

发表于 2020-03-02 更新于 2020-06-12 分类于 边缘计算
本文字数： 3.7k 阅读时长 ≈ 3 分钟

  1. 大网络使用Relu6很容易爆炸（Relu=max(0,x)）,而Relu6指的是min(max(0,x),6)。
    即控制最大值是6，为了在移动端设备float16的低精度的时候，也能有很好的数值分辨率。因为
    如果x很大的时候，则float16的精度无法描述这么大的精度值，带来精度损失。

  2. 全连接层就是用了很多个卷积核（大小为上一个卷积层大小），最后变成一个数，即没有了普通卷积层中卷积核滑动窗口的概念。

  3. op一般是值operation（操作）

  4. feature map是每一个filter对输入进行卷积后所有的输出，其中，feature map中的每个channel对应每个filter的输出。在某一个卷积层中，有多少个filter，feature map就有多少个channel。（但是很多也说一个特征图就对应一个channel，比如输出有64个channel，则就是说输出了64的特征图）

  5. 权值共享就是指在一个卷积过程中，输出的某一个feature map中的每一个点的权值都是共享的，也就是说，卷积形为滑动窗口时使用的weight都是一样的。

量化笔记

发表于 2020-03-02 更新于 2020-06-12 分类于 边缘计算
本文字数： 1.1k 阅读时长 ≈ 1 分钟

  1. 量化分为三种：一对权重进行量化，二是对特征图进行量化（神经元输出），对梯度量化（训练过程中）

  2. 对二值或三值网络进行训练时，通常做法是网络前馈过程为二值或三值，梯度更新过程为实数值。

  3. 和其它 bit 数稍高的量化网络（例如三值网络、2-bit 网络、4-bit 网络）相比，二值网络最突出的优点在于，1-bit 乘加操作可以通过位运算高效实现，因此可以无缝运行在主流硬件平台（x86、ARM）上，而三值、2-bit、4-bit 等等操作需要特殊硬件平台（FPGA、ASIC）支持。

  4. 量化的retrain：一般是把foward阶段把所有的权重值（w）量化后，再通过考虑梯度的量化（梯度近似）去拟合原来的量化函数，使得反向传播的时候更新值可以将量化函数考虑进去。将梯度量化后的值再对w进行更新，再使用量化函数，即w’=Q(w-α△w)。

  5. 在XNOR-net论文中，Binary Weight在卷积中的操作只涉及加减是因为，量化后的权重的符号与原来权重相同，所以做卷积操作的时候只是把所有输入的CHW的所有值取绝对值再加起来。

  6. 均匀量化的本质是用一个归一化因子将浮点数映射到它们最近的定点整数。

  7. XNOR-net网络的尺度因子α是一个浮点数，BP时使用二值化后的权重计算梯度，更新梯度值时却用原来的权重值减去用二值化后的权重梯度来进行更新。也就是说(Wt+1=Wt-lr*△W~t)。
    LQ-Net训练使用的尺度因子是一个向量v（K维，即用K个bit去量化。比如K=3，则v=[1,2,4]），这个v和量化后的值都是要优化的变量。在反向传递中，直接误差反向传播将是有问题的，因为量化函数的梯度几乎在任何地方都是0。所以BP过程为：对于激活，我们将量化函数的梯度设置为1（量化函数中输入的激活值激活值在q1和q2K时），其他地方为0；对于权重，梯度在任何地方都设置为1
    IR-Net训练的尺度因子是一个整数位移标量s。反向传播是通过逐步逼近反向传播中的符号函数来最小化梯度的信息损失。也就是通过近似sign函数，这个近似函数是连续的，可以直接求偏导数。原始近似越接近，则反向传播偏导时损失就会越小。
