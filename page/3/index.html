发表于 2020-04-05 更新于 2020-06-12 分类于 杂

  1. argparse.ArgumentParser 此模块用来解析命令行参数。也就是在终端执行一个python文件的时候
    （如：python test.py –resume –lr=0.1）后面的参数，一般会有默认值。
    参考：https://blog.csdn.net/vcvycy/article/details/88600134

  2. 如果网络的输入数据维度或类型上变化不大，设置 torch.backends.cudnn.benchmark = true 可以增加运行效率；
    如果网络的输入数据在每次 iteration 都变化的话，会导致 cnDNN 每次都会去寻找一遍最优配置，这样反而会降低运行效率。
    参考：https://www.pytorchtutorial.com/when-should-we-set-cudnn-benchmark-to-true/

  3. Dataloader方法产生一个可迭代对象(iterator)，每次输出指定batch_size大小的Tensor）。

  4. python中isnull函数的用法：
    https://blog.csdn.net/qq_40825479/article/details/83544430?depth_1-utm_source=distribute.pc_relevant.none-task-blog-BlogCommendFromBaidu-2&utm_source=distribute.pc_relevant.none-task-blog-BlogCommendFromBaidu-2

  5. python中类方法的属性需要加self，也就是self.xxx，这是方法的属性，可以通过实例化该类之后直接引用。
    类方法的变量不加self，也就是xxx，这个是方法的局部变量，不能被调用，只能在该方法内部使用！
    在方法中的self的变量可以看做是类内部的全局变量，可以被其他方法直接引用，方法中不加self的变量可以看成一个局部变量，该变量不能被直接引用。引用其他方法的变量时，也要用self。方法形参里面的的self其实代表了这个类的一个实例化，所以才可以去使用这个self去引用各个方法中的属性。

  6. nn.CrossEntropyLoss()内含了softmax，所以会看到最后一层都是nn.linear而不是softmax。

  7. “ / “就一定表示 浮点数除法，返回浮点结果;” // “表示整数除法
