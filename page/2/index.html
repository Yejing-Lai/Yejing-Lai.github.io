机器学习笔记

发表于 2019-11-02 更新于 2020-05-07 分类于 模型
本文字数： 712 阅读时长 ≈ 1 分钟

  1. 一个epoch就是对整个数据进行一次训练，比如在做梯度下降的时候，一个epoch就是对所有数据进行一次训练（梯度下降：全部数据集（BGD）一起训练，批次（MBGD），随机（一个个数据来训练SGD））

  2. Resnet就是加上跳跃链接：没两层加上一个捷径，构成一个残差快。
    参考：https://blog.csdn.net/qq_29893385/article/details/81207203

  3. 卷积后的长宽公式：output = (intput - filter_size + 2 x padding) / stride + 1
    pool后的长宽公式也是一样

  4. 多模态学习：比如图像的向量和文本的向量映射到同一个向量空间去。
    参考：https://zhuanlan.zhihu.com/p/53511144

  5. 最大化互信息条件就是最大化随机事件的相关性。在机器学习中，理想情况下，当互信息越大，说明两个事件的相关性越大，可以认为从数据集中拟合出来的分布与真是分布相同。

  6. Grid Search：一种调参手段；穷举搜索：在所有候选的参数选择中，通过循环遍历，尝试每一种可能性，表现最好的参数就是最终的结果。其原理就像是在数组里找最大值。（为什么叫网格搜索？以有两个参数的模型为例，参数a有3种可能，参数b有4种可能，把所有可能性列出来，可以表示成一个3*4的表格，其中每个cell就是一个网格，循环过程就像是在每个网格里遍历、搜索，所以叫grid search）
    参考：https://kknews.cc/code/aaaxmen.html

  7. RoI Pooling和RoI Align原理：https://www.cnblogs.com/wangyong/p/8523814.html

剪枝笔记

发表于 2019-11-02 更新于 2020-05-26 分类于 边缘计算
本文字数： 759 阅读时长 ≈ 1 分钟

  1. 剪枝过程中把其中的矩阵变成0之后，通过判断是否等于0，若等于0就跳过，不进行计算，这样就相当于是减少了FLOPs。

  2. 使用混合精度进行训练的时候。是指原始的weight是FP32的，将这个原始的weight转成FP16，通过前向传播，反向传播，获得FP16的梯度后。在进行更新的时候，使用原始的FP32的weight-FP16的梯度，最后更新的weight还是FP32的。这就是FP32和FP16进行混合训练，中间主要用了16位进行计算，在更新的时候再变成32位。

  3. 微调是经过剪枝后，去掉之前已经被被剪掉的连接（比如说mask），然后根据这个结构更新没有被剪掉的参数，修复精度。

模型压缩笔记

发表于 2019-11-02 更新于 2020-06-08 分类于 边缘计算
本文字数： 3.7k 阅读时长 ≈ 3 分钟

  1. 大网络使用Relu6很容易爆炸（Relu=max(0,x)）,而Relu6指的是min(max(0,x),6)。
    即控制最大值是6，为了在移动端设备float16的低精度的时候，也能有很好的数值分辨率。因为
    如果x很大的时候，则float16的精度无法描述这么大的精度值，带来精度损失。

  2. 全连接层就是用了很多个卷积核（大小为上一个卷积层大小），最后变成一个数，即没有了普通卷积层中卷积核滑动窗口的概念。

  3. op一般是值operation（操作）

  4. feature map是每一个filter对输入进行卷积后所有的输出，其中，feature map中的每个channel对应每个filter的输出。在某一个卷积层中，有多少个filter，feature map就有多少个channel。（但是很多也说一个特征图就对应一个channel，比如输出有64个channel，则就是说输出了64的特征图）

  5. 权值共享就是指在一个卷积过程中，输出的某一个feature map中的每一个点的权值都是共享的，也就是说，卷积形为滑动窗口时使用的weight都是一样的。
