linux笔记

发表于 2020-05-10 更新于 2020-06-12 分类于 杂
本文字数： 788 阅读时长 ≈ 1 分钟
  1. 改变目录名字: mv /home/lyj/abc /home/lyj/aaa
  2. 清空终端屏幕快捷键:ctrl+L
  3. 修改配置文件（相当于Windows中的环境变量），要改两处地方，参考：https://blog.csdn.net/hang916/article/details/79530108
  4. watch -n 10 nvidia-smi ：设置为每 10s 显示一次显存的情况
  5. nohup挂起进程在后台运行，可以看nohup.out文件看输出信息
  6. 在linux中可以通过wget url下载一个tar.gz的文档。并使用tar -zxvf xxxx.tar.gz解压文档到当前目录。
  7. rm -f xxx 可以删除某个文件。rm -rf xxx删除某个文件夹。
  8. pwd显示当下的绝对目录
  9. nohup python -u trainer.py >out.log 2>&1 & 使用nohup运行后台程序，注意使用nohup之后要正常退出终端（输入exit()）
    任何nohup命令后都要有 &;然后再回车键再输入exit()
  10. conda env list 或 conda info -e 查看当前存在哪些虚拟环境
  11. conda create -n your_env_name python=X.X（2.7、3.6等)
    删除虚拟环境。
    使用命令conda remove -n your_env_name(虚拟环境名称) –all，
  12. 在当前目录下查找某个文件:find . -name “klibc”
    如果不明确文件名称，可以使用通配符:find . -name “libc“
  13. ps -ef | grep nginx 查看现有的进程
  14. tar -xvf file.tar //解压 tar包
    tar -xzvf file.tar.gz //解压tar.gz
  15. ps -ef：显示所有的进程号
    ps pid号码：显示该进程pid的所有信息
    ps -ef|grep 关键字：返回所有包含关键字的进程，关键字就是一个正则表达式
