模型压缩笔记

发表于 2019-11-02 更新于 2020-04-01 分类于 边缘计算
本文字数： 1.6k 阅读时长 ≈ 1 分钟

  1. 大网络使用Relu6很容易爆炸（Relu=max(0,x)）,而Relu6指的是min(max(0,x),6)。
    即控制最大值是6，为了在移动端设备float16的低精度的时候，也能有很好的数值分辨率。因为
    如果x很大的时候，则float16的精度无法描述这么大的精度值，带来精度损失。

  2. 全连接层就是用了很多个卷积核（大小为上一个卷积层大小），最后变成一个数，即没有了普通卷积层中卷积核滑动窗口的概念。

  3. op一般是值operation（操作）

  4. feature map是每一个filter对输入进行卷积后所有的输出，其中，feature map中的每个channel对应每个filter的输出。在某一个卷积层中，有多少个filter，feature map就有多少个channel。（但是很多也说一个特征图就对应一个channel，比如输出有64个channel，则就是说输出了64的特征图）

  5. 权值共享就是指在一个卷积过程中，输出的某一个feature map中的每一个点的权值都是共享的，也就是说，卷积形为滑动窗口时使用的weight都是一样的。

  6. FLOPS：注意全大写，是floating point operations per second的缩写，意指每秒浮点运算次数，理解为计算速度。是一个衡量硬件性能的指标。
    FLOPs：注意s小写，是floating point operations的缩写（s表复数），意指浮点运算数，理解为计算量。可以用来衡量算法/模型的复杂度。
    时间复杂度可用FLOPs来衡量，空间复杂度（params）包含两个部分：总参数量（模型体积）+各层输出特征图。
    计算着两个东西可参考：https://zhuanlan.zhihu.com/p/31575074

  7. dropout中是把临时以一定的概率p删掉（隐藏掉，并不是把其真的删掉，就是在某个batch过程中把这些神经元的激活值设为0）一些（一般来说是一半）的隐层神经元，在不同的batch里面置为0的神经元都不同。在测试阶段，每个神经单元的权重参数都要乘以概率p。参考：https://blog.csdn.net/program_developer/article/details/80737724

  8. 全局平均池化（GAP）：一般用来替换掉全连接层，减少参数。比如在VGG-16分类任务中，如果最后要输出10个类，则直接在最后一层输出有10个channel（特征图），将这10个channel的所有值平均，得到十个数，再用softmax进行分类。

  9. non-structure和structure有各自的好处和不同。filter(channel pruning)和layer pruning都是结构化剪枝，改变值的都是非结构话剪枝。

  10. pool层（也叫下采样，降采样）是在做卷积的过程中同时运行的，比如输入224×224×64，先经max pooling（最大化池化），滤波器为2x2，步长为2，图像尺寸减半，池化后的尺寸变为112x112x64，然后再经128个3x3的卷积核，卷积，ReLU激活，尺寸变为112x112x128。
    VGG16的详解参考：https://blog.csdn.net/weixin_43496706/article/details/101210981?depth_1-utm_source=distribute.pc_relevant.none-task&utm_source=distribute.pc_relevant.none-task
    pool层作用的详解：https://blog.csdn.net/zxyhhjs2017/article/details/78607469

  11. ResNet：网络深度增加，准确度下降，这并不是过拟合的问题（因为层数增加的时候训练误差和测试误差都降低了，但是过拟合是训练误差高，而测试误差降低）。主要是学习恒等映射F(x)+x，x是输入
